    Слайд 1. Титульный

Добрый день уважаемая комиссия.

Тема моей работы: Метод разбиения категориальных данных на основе агломеративного подхода иерархической кластеризации.


    Слайд 2. Актуальность

Иерархическая кластеризация активно применяется в различных областях. Однако, данный метод имеет существенный недостаток. Он объединяет равноудаленные обособленные кластеры попарно в несколько этапов. Что уменьшает компактность и разделенность кластеров.

Для устранения данного недостатка было принято решение разработать гибридный метод кластеризации.


    Слайд 3. Цель и задачи

Целью моей работы является разработка метода разбиения категориальных данных на основе агломеративного подхода иерархической кластеризации.

Для достижения поставленной цели требуется решить задачи, представленные на слайде.


    Слайд 4. Выбор метода разбиения для уточнения элементов в кластерах

На данном слайде представлено сравнение методов кластеризации по типу обрабатываемых признаков, типу алгоритма, входным и выходным данным.

Иерархические методы возвращают бинарное дерево кластеров, узлы которого содержат номера объектов, входящих в данную группу. На основе этих узлов можно вычислить центры кластеров, используемые в центроидных методах разбиения.

Поэтому именно кластеризация центроидного типа может быть использованы для уточнения результатов иерархического разбиения.

Чтобы разрабатываемый метод мог обрабатывать как категориальные, так и числовые признаки, для уточнения результатов был выбран метод k-прототипов.


    Слайд 5. Метод разбиения на основе иерархической кластеризации

Реализация гибридного метода кластеризации состоит из четырех основных этапов:
-- построение матрицы несходства;
-- применение агломеративного подхода иерархической кластеризации;
-- получение списка номеров объектов, входящих в каждый кластер;
-- уточнение принадлежности элементов к группам с помощью метода кластеризации k-прототипов.


    Слайд 6. Выбор критерия связи кластеров

В иерархической части алгоритма встает вопрос, как вычислять расстояния между кластерами. Для решения этой задачи используются различные критерии связи (одиночная связь, полная и средняя).

Для гибридного метода была выбрана полная связь, которая вычисляет расстояние между двумя кластерами как самое длинное расстояние между двумя точками в каждой группе.

Полная связь была выбрана, так как она обеспечивает более компактные кластеры, чем одиночная или средняя.


    Слайд 7. Сравнение мер расстояний

Сравнение мер расстояний проводилось по возможности обрабатывать числовые и категориальные данные.

Для вычисления расстояний между объектами в гибридном методе разбиения было выбрано расстояние Говера, так как оно едиственное из рассмотренных может обрабатывать оба типа признаков.


    Слайд 8. Вычисление расстояния Говера

Суть метода Говера заключается в вычислении для каждого признака своей оценки сходства в диапазоне от 0 до 1, после чего, расстояние между объектами считается как сумма всех таких оценок, деленная на число признаков.

Оценка сходства считается по разному для двух типов переменных:
-- для числовых она вычисляется как модуль разности признаков, деленный на диапазон значений данного признака;
-- для категориальных оценка равняется 0, если признаки совпали и 1 в противном случае.


    Слайд 9. Гибридный метод кластеризации

Идея разработанного метода кластеризации заключается в объединении иерархического метода с методом k-прототипов. Такой подход позволил избавиться от недостатков обоих методов.

С помощью уточнения результатов разбиения удалось добиться добавления обособленных объектов, равноудаленных друг от друга, в один кластер. Также удалось избежать рандомной инициализации центроидов, за счет использования узлов бинарного дерева кластеров, полученного в результате работы иерархической части метода.


    Слайд 10. Нахождение матрицы несходства

На первом этапе гибридного метода происходит вычисление матрицы несходства, которая показывает степень различия между объектами. Для определения расстояния между элементами матрицы используется расстояние Говера.


    Слайд 11. Иерархическая часть гибридного метода

Так как иерархический метод имеет агломеративный подход, то изначально каждый объект размещается в отдельный кластер. После чего ближайшие группы начинают объединяться, постепенно образовывая бинарное дерево.


    Слайд 12. Центроидная часть гибридного метода

На данном этапе происходит уточнение принадлежности элементов к кластерам с помощью метода k-прототипов.

Сначала вычисляются центры кластеров на основе списков номеров объектов, входящих в каждую группу. Далее для каждого объекта определяется ближайший к нему центр кластера. После чего находятся новые центроиды. Шаги повторяются до тех пор, пока центры не перестанут изменяться.


    Слайд 13. Результаты разбиения для 30 кластеров

На данном слайде представлены результаты разработанного мной гибридного метода разбиения в сравнении с методом k-прототипов. Точками на графиках показаны расстояния от объектов до центров их кластеров. Чем больше объектов находится на меньшем расстоянии до центров, тем лучше. В данном случае визульно сложно оценить, какой метод показывает лучшие результаты. Для этого существуют специальные методы оценки качества кластеризации, которые будут представлены далее.


    Слайд 14. Результаты разбиения для 60 кластеров

При 60 итоговых кластерах разница между методами становится очевиднее. Гибридный метод показывает лучшие результаты. Это связано с рандомной инициализацией начальных центроидов в методе k-прототипов. В гибридном алгоритме центроиды вычисляются на основе кластеров, полученных в результате работы иерархической части метода.


    Слайд 15. Оценка качества кластеризации (метод локтя)

Известны несколько методов оценки качества кластеризации. Метод локтя показывает среднее расстояние между объектами внутри групп и используется в тех случаях, когда важна компактность кластеров.

По результатам сравнения можно сделать вывод о том, что гибридный метод разбиения надо применять в тех случаях, когда метод k-прототипов показывает лучшие результаты в сравнении с иерархическим. 

Это происходит, когда в иерархическом методе попарно объединяются равноудаленные кластеры.


    Слайд 16. Оценка качества кластеризации (метод оценки силуэтов)

Метод оценки силуэтов показывает, насколько близко каждая точка внутри одной группы расположена к точкам ближайшего соседнего кластера.

Значения коэффициентов силуэтов находятся в диапазоне от −1 до 1. Чем ближе значение к 1, тем лучше разделен кластер.

Метод k-прототипов показал наихудшие результаты. Это связано с рандомной инициализацией начальных центроидов. Из-за чего центры кластеров могли образовываться рядом друг с другом.

Гибридный метод показал лучшие результаты в сравнении с иерархическим, когда объединялись равноудаленные кластеры.
